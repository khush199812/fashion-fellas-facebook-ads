# Facebook Ads Practice Campaign – Fashion Fellas

## Project Overview
This project is a Facebook Ads practice campaign created for a fictional men's clothing brand called Fashion Fellas. The objective of this project is to demonstrate understanding of Meta Ads campaign structure, targeting, creatives, and strategy.

## Business Description
Fashion Fellas is a fictional menswear brand focused on modern, everyday clothing for men aged 18–35.

## Campaign Objective
Brand Awareness

## Target Audience
- Gender: Men
- Age: 18–35
- Location: India
- Interests: Fashion, lifestyle, menswear

## Ad Format
- Image Ads
- Instagram & Facebook placements

## Destination
Messaging apps (Instagram & Messenger)

## Expected Outcome
- Increased brand awareness
- Engagement and profile visits

## Note
This is a practice project created for learning purposes and does not involve real ad spend.
